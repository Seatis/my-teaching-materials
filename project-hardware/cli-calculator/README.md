### Print usage
 - At application startup the following message should be shown:

```
Calculator
====================
Commands:
 +  summation
 -  subtraction
 *  multiplication
 /  division
 ^  squaring
 <  root
 %  division with remainder
 ```

### +

For example the content is:

```
3 + 2 = 5

```

### -


For example the content is:

```
3 - 2 = 1

```

### *

For example the content is:

```
3 * 2 = 6

```

### /

For example the content is:

```
6 / 2 = 3

```

### ^

For example the content is:

```
3 ^ 2 = 9

```

### <

For example the content is:

```
< 4 = 2

```

### %

For example the content is:

```
3 % 2 = 1,5

```
