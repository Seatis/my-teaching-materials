## Material Review
- naming variables
  - intention-revealing
  - avoid disinformation
  - pronounceable
  - searchable
- naming classes
  - noun or noun phrase
  - not a verb!
  - avoid data synonyms
- naming class instance methods
  - verb or verb phrase
  - don't be cute
  - be consistent
  - don't pun (2 meanings)
  - context
