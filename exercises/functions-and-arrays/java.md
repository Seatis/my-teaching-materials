## Material Review
- Declaring
  - constants
  - initial values of arrays `{}`
  - `new` keyword
- Functions
  - void or return a value
  - can be used to compartmentalise code (make more readable)
  - have own variables
  - take parameters
- Arrays
  - can be measured (have `length`)
  - can be iterated through
  - type definition (e.g. `String[]`)
  - toString
  - copyOf
  - sort
  - binarySearch
  - foreach statement
